Thank you for trying Beobachter, your new tool to watch at files.

If you want to run the application, no matter the operating system you are, double click the Beobachter.jar.
If you don't see anything, open a console/terminal and type "java -jar Beobachter.jar" to check the messages printed by the application.

IMPORTANT: You won't execute this tool without the Java runtime (JRE) v1.5 installed. Visit http://java.sun.com to start downloading the Java software.
